# Images

Drop your own photos into this folder to replace the grey placeholders.
Simply save your photo with the exact same filename — the site will use it automatically.

## How to replace an image

1. Take your photo and save/export it as a JPEG
2. Rename it to match the filename in the table below
3. Drop it into the correct subfolder (replacing the existing placeholder)
4. Refresh the browser — done!
